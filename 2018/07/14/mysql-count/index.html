MySQL count(常量)/count（*）/count（特定列名的区别）

🍊AwfulOrange
2018-07-14
MySQL

背景

在实际开发过程中，需要计算当前条件下有多少数据时，都需使用count()方法来获取行数，同事中有各种count（）的方法，这里探讨一下常见几种count的区别

表结构

各种count的结果

select * from tb_collect_card_info_copy1;

select count(*) from tb_collect_card_info_copy1;

select count(app_id) from tb_collect_card_info_copy1;

select count(consumer_id) from tb_collect_card_info_copy1;

select count(1) from tb_collect_card_info_copy1;

结论

count(列名) 时count 会忽略为null的列
count(*)、count(1)、count(CONSTANT) 即传统意义上的行数
